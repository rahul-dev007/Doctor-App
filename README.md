# ЁЯй║ Doctor-App

A full-stack **Doctor Appointment Booking Application** built using **Next.js (frontend)**, **Express (backend)**, and **MongoDB (database)**.  
This project includes **secure authentication**, **role-based access**, and **profile management**.

## ЁЯЪА Features
- ЁЯФС User authentication with JWT (Login, Register, Logout)
- ЁЯСд Profile management with image upload
- ЁЯй║ Role-based access (User, Doctor, Admin)
- ЁЯУЕ Appointment booking system (Upcoming feature)
- ЁЯЫбя╕П Cookie-based session handling for security

---

## ЁЯЫая╕П Tech Stack
- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB + Mongoose
